# 07/04
Implemented `draw::line` using DDA line drawing and got myself a ***super deluxe wireframe cube***. 

Was wondering between these two:

```cpp
void line(graphics::Context&, int x0, int y0, int x1, int y1, uint32_t color);
void line(graphics::Context&, Vec2i x, Vec2i y, uint32_t color);
```

With the second, I'd need to create `Vec2i` or use generics. `Vec2i` might be  preferred, glm has [ivec2](https://cglm.readthedocs.io/en/latest/ivec2.html) instead of using generics. 

For lines both SDL and Raylib use the individual cords, but for rect Raylib uses x, y, w, h separately while SDL has `SDL_rect`.
